# Heladería de Juana - Brand Spec

## Visual DNA
- **Core Style:** Nintendo Cozy (Animal Crossing / Kirby) meets Studio Ghibli.
- **Atmosphere:** Bubbly, soft, emotional, and handmade.
- **Key Traits:** High rounding (24px+), squishy buttons, subtle inner glows, and pastel gradients.

## Color Palette (OKLch)
- `--bg`:      oklch(97% 0.018 70)   /* Creamy Vanilla */
- `--surface`: oklch(99% 0.008 70)   /* Soft Paper */
- `--fg`:      oklch(25% 0.04 40)    /* Warm Chocolate (Text) */
- `--accent`:  oklch(75% 0.15 20)    /* Strawberry Pink */
- `--mint`:    oklch(92% 0.08 160)   /* Fresh Mint */
- `--sky`:     oklch(90% 0.06 230)   /* Clear Sky Blue */
- `--border`:  oklch(90% 0.02 70)    /* Soft Cookie Border */

## Typography
- **Display:** 'Fredoka One', 'Varela Round', system-ui, sans-serif (Rounded, bubbly)
- **Body:** 'Varela Round', 'Inter', system-ui, sans-serif (Friendly, legible)

## Layout & Posture
- **Rounding:** 24px default for cards, 50% (circles) for icons.
- **Shadows:** Soft inner glows (white 40%) and diffused drop shadows (fg at 8%).
- **Borders:** 2px-3px dashed or solid with 'soft cookie' colors.
